# Traffic Light 4 Arah

Program Traffic Light 4 arah ini dibuat menggunakan tinkercad. penjelasan singkat program ini : <br>
1. Terdapat 4 Arah yaitu Barat, Utara, Timur dan Selatan.
2. Setiap arah mempunyai 3 LED yaitu merah, kuning dan hijau.
3. Kondisi default semua lampu dalam kondisi merah sebelum satu sisi aktif
4. Lampu hijau menyala bergiliran searah jarum jam : Barat - Utara - Timur - Selatan.
5. Lampu hijau menyala 5 detik, Lampu kuning ada efek kedip 3 kali dan menyala 2 detik setelah berkedip dan lampu merah menyala setelah hijau selesai
